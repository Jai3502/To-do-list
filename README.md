# To-do-list
**Description:** This Python-based To-Do List application allows users to efficiently manage daily tasks through a simple console interface. Users can add, view, and remove tasks easily. It helps improve productivity by keeping track of pending work, ensuring better organization and time management.
#  Python To-Do List Application

This project is a simple command-line **To-Do List application** written in Python.  
It allows users to add tasks, view all tasks, and remove completed tasks easily through an interactive menu.

---

##  Project File

**jp.py**  
A Python script containing the full implementation of the To-Do List application.

##  Features

-  Add new tasks  
-  View all saved tasks  
-  Remove tasks by selecting task number  
-  Simple menu-driven interface  
-  Beginner-friendly Python example
